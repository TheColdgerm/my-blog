## 一、求导公式

### 1. 基本初等函数的导数
- 常数：$(C)' = 0$
- 幂函数：$(x^\mu)' = \mu x^{\mu-1}$（$\mu$ 为实数）
- 指数函数：$(e^x)' = e^x$，$(a^x)' = a^x \ln a$
- 对数函数：$(\ln x)' = \frac{1}{x}$，$(\log_a x)' = \frac{1}{x \ln a}$
- 三角函数：
  - $(\sin x)' = \cos x$
  - $(\cos x)' = -\sin x$
  - $(\tan x)' = \sec^2 x$
  - $(\cot x)' = -\csc^2 x$
  - $(\sec x)' = \sec x \tan x$
  - $(\csc x)' = -\csc x \cot x$
- 反三角函数：
  - $(\arcsin x)' = \frac{1}{\sqrt{1-x^2}}$
  - $(\arccos x)' = -\frac{1}{\sqrt{1-x^2}}$
  - $(\arctan x)' = \frac{1}{1+x^2}$
  - $(\operatorname{arccot} x)' = -\frac{1}{1+x^2}$

### 2. 求导法则
- **线性法则**：$(u \pm v)' = u' \pm v'$，$(cu)' = c u'$
- **乘积法则**：$(uv)' = u'v + uv'$
- **商法则**：$\left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2}$（$v \neq 0$）
- **链式法则**：$[f(g(x))]' = f'(g(x)) \cdot g'(x)$
- **反函数求导**：若 $y = f(x)$ 可导且 $f'(x) \neq 0$，则反函数 $x = \varphi(y)$ 的导数为 $\varphi'(y) = \frac{1}{f'(x)}$ 或 $\frac{dx}{dy} = \frac{1}{dy/dx}$
- **隐函数求导**：由 $F(x,y)=0$ 确定 $y=y(x)$，则 $\frac{dy}{dx} = -\frac{F_x}{F_y}$（$F_y \neq 0$）
- **参数方程求导**：$\begin{cases} x = x(t) \\ y = y(t) \end{cases}$，则 $\frac{dy}{dx} = \frac{y'(t)}{x'(t)}$，二阶导 $\frac{d^2y}{dx^2} = \frac{x'(t)y''(t) - y'(t)x''(t)}{[x'(t)]^3}$

## 二、积分公式

### 1. 不定积分基本公式（与导数对应）
- $\int k \, dx = kx + C$
- $\int x^\mu \, dx = \frac{x^{\mu+1}}{\mu+1} + C$（$\mu \neq -1$）
- $\int \frac{1}{x} \, dx = \ln|x| + C$
- $\int e^x \, dx = e^x + C$
- $\int a^x \, dx = \frac{a^x}{\ln a} + C$
- $\int \sin x \, dx = -\cos x + C$
- $\int \cos x \, dx = \sin x + C$
- $\int \sec^2 x \, dx = \tan x + C$
- $\int \csc^2 x \, dx = -\cot x + C$
- $\int \sec x \tan x \, dx = \sec x + C$
- $\int \csc x \cot x \, dx = -\csc x + C$
- $\int \tan x \, dx = -\ln|\cos x| + C$
- $\int \cot x \, dx = \ln|\sin x| + C$
- $\int \sec x \, dx = \ln|\sec x + \tan x| + C$
- $\int \csc x \, dx = \ln|\csc x - \cot x| + C$
- $\int \frac{1}{\sqrt{1-x^2}} \, dx = \arcsin x + C$（或 $-\arccos x + C$）
- $\int \frac{1}{1+x^2} \, dx = \arctan x + C$（或 $-\operatorname{arccot} x + C$）
- $\int \frac{1}{\sqrt{a^2-x^2}} \, dx = \arcsin\frac{x}{a} + C$（$a>0$）
- $\int \frac{1}{a^2+x^2} \, dx = \frac{1}{a} \arctan\frac{x}{a} + C$
- $\int \frac{1}{x^2-a^2} \, dx = \frac{1}{2a} \ln\left|\frac{x-a}{x+a}\right| + C$
- $\int \frac{1}{\sqrt{x^2 \pm a^2}} \, dx = \ln\left|x + \sqrt{x^2 \pm a^2}\right| + C$

### 2. 积分运算法则
- **线性性质**：$\int [f(x) \pm g(x)] \, dx = \int f(x) \, dx \pm \int g(x) \, dx$，$\int k f(x) \, dx = k \int f(x) \, dx$
- **分部积分法**：$\int u \, dv = uv - \int v \, du$（选 $u$ 遵循“反对幂三指”顺序）
- **换元积分法**：
  - *第一类换元（凑微分）*：$\int f(\varphi(x)) \varphi'(x) \, dx = \int f(u) \, du$，令 $u = \varphi(x)$
  - *第二类换元*：$\int f(x) \, dx$，令 $x = \psi(t)$，则 $dx = \psi'(t) \, dt$，转化为 $\int f(\psi(t)) \psi'(t) \, dt$，常用于去根号（三角代换、倒代换等）

### 3. 定积分常用公式与性质
- **牛顿-莱布尼茨公式**：$\int_a^b f(x) \, dx = F(b) - F(a)$，其中 $F'(x) = f(x)$
- **线性性质**：同不定积分
- **换元法**：$\int_a^b f(x) \, dx = \int_\alpha^\beta f(\varphi(t)) \varphi'(t) \, dt$，需换限
- **分部积分法**：$\int_a^b u \, dv = [uv]_a^b - \int_a^b v \, du$
- **奇偶对称性**：
  - 若 $f(x)$ 为偶函数，则 $\int_{-a}^a f(x) \, dx = 2 \int_0^a f(x) \, dx$
  - 若 $f(x)$ 为奇函数，则 $\int_{-a}^a f(x) \, dx = 0$
- **周期性**：若 $f(x)$ 以 $T$ 为周期，则 $\int_a^{a+T} f(x) \, dx = \int_0^T f(x) \, dx$
- **华里士公式（点火公式）**：
  - $\int_0^{\pi/2} \sin^n x \, dx = \int_0^{\pi/2} \cos^n x \, dx = \begin{cases} \frac{(n-1)!!}{n!!} \cdot \frac{\pi}{2}, & n \text{为正偶数} \\ \frac{(n-1)!!}{n!!}, & n \text{为正奇数} \end{cases}$
- **区间再现公式**：$\int_a^b f(x) \, dx = \int_a^b f(a+b-x) \, dx$
- **欧拉积分**：$\int_0^{\pi/2} \ln\sin x \, dx = -\frac{\pi}{2} \ln 2$

## 三、多元函数微积分常用公式（补充）

- **偏导数**：$\frac{\partial z}{\partial x}$，$\frac{\partial z}{\partial y}$（$z = f(x,y)$）
- **全微分**：$dz = \frac{\partial z}{\partial x} \, dx + \frac{\partial z}{\partial y} \, dy$
- **梯度**：$\operatorname{grad} f = \left( \frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}, \frac{\partial f}{\partial z} \right)$
- **多元复合函数求导（链式法则）**：如 $z = f(u,v)$，$u = \varphi(x,y)$，$v = \psi(x,y)$，则 $\frac{\partial z}{\partial x} = \frac{\partial f}{\partial u} \frac{\partial u}{\partial x} + \frac{\partial f}{\partial v} \frac{\partial v}{\partial x}$
- **隐函数求导**：由 $F(x,y,z)=0$ 确定 $z = z(x,y)$，则 $\frac{\partial z}{\partial x} = -\frac{F_x}{F_z}$，$\frac{\partial z}{\partial y} = -\frac{F_y}{F_z}$
- **重积分换元**：$\iint_D f(x,y) \, dxdy = \iint_{D'} f(x(u,v), y(u,v)) \, |J| \, du dv$，其中 $J = \frac{\partial(x,y)}{\partial(u,v)}$
- 曲线积分、曲面积分等相关公式（如格林公式、高斯公式、斯托克斯公式）在此略去，可根据需要另行总结。
